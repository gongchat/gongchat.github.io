Welcome to Gong

Gong is an open source XMPP chat client. It is currently in alpha release.

Customize All the Colors

Gong lets you customize the colors within the application so you can have a chat client with your favorite colors.

{% include mini-gong.html %}

Customize your Font

Change your font size and use any font available from Google Fonts

Media Support

Get previews for messages with images, gifs, YouTube and GetYarn

Markdown Support

Gong supports markdown. This includes text formatting and emojis!
